Git Commands:
git branch // Displays all the branches of this repository
git branch <name> // Makes a branch with <name> as the branch name
git checkout <name> // Changes to the <name> branch
git fetch // Gets the remote master and updates yours to match (keeps you on the same spot)

CSS Selectors:
a, b Multiple Element Selector
a b Descendant Selector
a > b Child Selector
a + b Adjacent Sibling Selector
[a=b] Attribute Selector
a:b Psuedoclass Selector
a::b Pseudoelement Selector
